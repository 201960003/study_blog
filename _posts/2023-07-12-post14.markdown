오늘은 플러터에서 파이어베이스를 사용하면서 나오는 에러에 대해서 공부해 보려고 합니다.<br>
무엇이냐면? 바로바로~~~ 

```
 [firebase_database/permission-denied] Client doesn't have permission to access the desired data.
```

라는 에러입니다. <br>

```
{
  "rules": {
    ".read": true,
	".write": true
  }
}
```

이 에러는 realtime database의 규칙을 위와 같이 수정한다면 간단히 해결할 수 있습니다.
<br><br><br>
