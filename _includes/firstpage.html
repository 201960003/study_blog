# 카테고리별 포스트 보기 헤더의 카테고리 메뉴를 통해 원하는 카테고리로 이동할 수 있습니다. ### 주요 카테고리 - **Javascript** - JavaScript 관련 포스트 - **Flutter** - Flutter 관련 포스트 - **Java** - Java 관련 포스트 - **HTML/CSS** - HTML, CSS 관련 포스트 - **Node.js** - Node.js 관련 포스트 - **Git** - Git 관련 포스트 - **Firebase** - Firebase 관련 포스트 - **전체** - 모든 포스트 보기 --- > **팁**: 헤더의 카테고리 메뉴를 클릭하면 해당 카테고리의 포스트만 볼 수 있습니다.
